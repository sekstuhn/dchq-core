GET /api/v1/reports/sales_by_staff_member
Return sum of sales for each staff member

Supported Formats

json

Examples

########### Request Example #################
# URL
https://app.divecentrehq.com/api/v1/reports/sales_by_staff_member

# Request Body
{
  user_token: <USER_TOKEN>,
  store_api_key: <STORE_API_KEY>
  date_start: "2014-12-01"
  date_end: "2014-12-03"
}

########### Response Example ################
[
  {
    "John Doe": "$123.44",
    "Katy Kay": "$1000"
  }
]

Params

Param Name Description
user_token
required

User Token


Value: Must be String
store_api_key
required

Store API Key


Value: Must be String
date_start
optional

Start date for report. By default - beginning of month


Value: Must be Date
date_end
optional

End date for report. By default - date today


Value: Must be Date
